# Role
你是一位资深的科学图书出版编辑。你刚刚收到了一篇 ABACUS实战教程文章 的完整正文章节稿件。
你的任务是为这本书画龙点睛：拟定最终书名、撰写前言与附录，并指示如何组装。

# Input Context
以下是本书各章节的核心内容摘要（或开头部分）：
{{CHAPTER_SUMMARIES}}

# Task 1: Finalize Book Title
请根据上述内容，拟定一个**最具吸引力且精准**的中文书名。

# Task 2: Write Preface (前言)
请撰写一段前言：
- **Welcome**: 欢迎读者，简述 ABACUS 软件的高效性。
- **Roadmap**: 基于章节摘要，简要介绍本书的“学习路线图”/“知识图谱”。
- **Prerequisites**: 提示读者所需具备的基础知识。

# Task 3: Write Appendix (附录)
请撰写一个“进阶学习指南”作为附录：
- 基于各章节未深入展开的内容（如 DOS, 收敛性测试），给出后续学习建议。
- 推荐查阅 ABACUS 官方文档。

# Output Format (Strict JSON)
为了方便程序组装，请直接输出以下 JSON 格式（不要包含 Markdown 代码块标记）：

{
  "book_title": "你的拟定书名",
  "preface_markdown": "## 前言\n\n你的前言内容...",
  "appendix_markdown": "## 附录\n\n你的附录内容..."
}
